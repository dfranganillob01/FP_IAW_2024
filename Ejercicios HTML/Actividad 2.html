Cabecera

Charset

Charset es un valor que denota la codificación de caracteres que será usada en un documento HTML.

Autor

El elemento head de un documento HTML es la parte
que no se muestra en el navegador cuando se carga la página.

Description

El tag de meta description (meta descripción) es un tag de código HTML que permite que una página web tenga una descripción.

Keywords

Las palabras clave o keywords son términos que se utilizan
para realizar consultas en Google u otro buscador con el objetivo de acceder a una lista de sitios web que tengan una relación directa con lo que buscamos.
